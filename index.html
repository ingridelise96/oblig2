Math ML

Math ML bruker XML til å lage matematiske formler som både er visuelt bedre oppsatt og lettere å forstå enn om man skulle skrevet dem med HTML. Klikk på overskriften for å lese mer om Math ML.

SVG

SVG står for Scalable Vector Graphics og er vektorfigurer som kan lages med XML-kode. SVG mister ikke kvaliteten om det skaleres opp eller ned i størrelse, Klikk på overskriften for å lese mer om SVG.
